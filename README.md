# PopX UI Design Assignment

This project is a pixel-perfect, mobile-centered React application built from an Adobe XD/Figma design file. It was developed as a qualifier task for an interview.

## 🚀 Live Demo
* **Live Site:** [PopX Assignment](https://popx-assignment-topaz.vercel.app/)
* **Design File:** [Adobe XD Design Link](https://xd.adobe.com/view/b68eea25-003d-4a5d-8fdd-d463eeb20b32-e3dd)

## 🛠️ Built With
* React.js
* React Router v6 (for seamless navigation)
* Custom CSS (for pixel-perfect styling)

## 📱 Features
* **Mobile-First Design:** The application interface is centered on the webpage, matching the mobile app design requirements.
* **Authentication Flow:** Includes smooth navigation between the Landing, Login, Registration, and Profile pages.
* **Password Toggle:** Custom password visibility toggle (eye icon) implemented on both Login and Registration forms.

## 💻 Getting Started
To run this project locally:

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
